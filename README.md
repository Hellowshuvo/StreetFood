# 🚀 Street Food

Map-based web app to discover and rate street food stalls in Bangladesh.

## ✨ Features
- 🗺️ Map discovery
- ⭐ Ratings & reviews
- 📍 Location-based browsing
- 📱 Mobile-first UI

## 🛠️ Stack
- Next.js 14
- Supabase
- PostgreSQL
- Leaflet
- Tailwind CSS

## 🚀 Run Locally
```bash
git clone https://github.com/Hellowshuvo/street-food.git
cd street-food
npm install
npm run dev
