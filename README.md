🌮 StreetFood

StreetFood is a map-first web app to discover, rate, and share street food spots across Bangladesh. It works like a mini social platform where users explore stalls, leave reviews, and post experiences.

🚀 Features
📍 Interactive map-based discovery
⭐ Ratings & reviews
📰 Feed for user posts & updates
🌗 Dark / Light mode
🛠 Tech Stack

Next.js • Supabase • Leaflet • Tailwind CSS

🎯 Goal

Build the go-to platform for street food discovery powered by real users.
